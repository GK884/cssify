## Installation

Use the package manager [npm](https://www.npmjs.com/) to install CSSify.

For the server:
```bash
git clone <path/to/git>
cd cssify/backend
npm install 
```
For the client
```bash
git clone <path/to/git>
cd cssify/client
npm install 
```

## Usage

Combined server for entire application:
```bash
cd cssify/backend
npm run dev
```
